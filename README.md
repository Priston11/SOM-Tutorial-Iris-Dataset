# Self-Organizing Map (SOM) Tutorial with Iris Dataset

## Overview
This repository contains a complete tutorial on Self-Organizing Maps (SOMs), demonstrating unsupervised learning and data visualization using the Iris dataset.

## Contents
- `figures/` - All generated figures (U-Matrix, label overlay, component planes, cluster map)
- `som_tutorial.ipynb` - Jupyter Notebook containing full code
- `webpage/` - HTML tutorial ready for submission
- `README.md` - This file

## Features
- Manual SOM implementation using NumPy
- U-Matrix and component planes for feature visualization
- Cluster assignment map
- Colorblind-friendly visualizations
- Accessible web tutorial with alt text
